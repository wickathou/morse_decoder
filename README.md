# 📖 [Morse-Decoder] <a name="about-project"></a>

**[Morse-Decoder]** is a microapp that decodes morse strings into plain language.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

**Ruby**

### Key Features <a name="key-features"></a>

- **[Function to decode individual characters]**
- **[Functions to decode words]**
- **[Functions to decode complete sentences]**

## 💻 Getting Started <a name="getting-started"></a>


### Prerequisites

In order to run this project you need:

- Ruby

### Usage

- To run the app,
  - Open a bash terminal
  - `cd` into the folder containing the file **morseDecoder.rb**
  - Run the command `irb morseDecoder.rb`
